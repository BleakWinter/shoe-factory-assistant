# shoe-factory-assistant

鞋厂订单与打印助手系统，第一阶段聚焦订单原稿上传、Excel 订单识别、PDF 预览和打印任务创建。

## 项目结构

```text
shoe-factory-assistant
├── backend
├── web
├── print-agent
├── sql
└── docs
```

## 模块说明

- `backend`: Spring Boot 后端服务。
- `web`: React + Vite + Ant Design 前端页面工程。
- `print-agent`: 第二阶段本地打印代理服务。
- `sql`: 数据库初始化和业务表脚本。
- `docs`: 技术、接口和部署文档。

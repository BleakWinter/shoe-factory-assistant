# 技术文档

## 项目概览

`shoe-factory-assistant` 第一阶段实现订单原稿上传、Excel 订单识别、订单列表过滤、按打印类型生成 PDF 预览、确认后创建打印任务。

## 模块划分

- `backend`: 负责原稿保存、订单识别、订单查询、PDF 预览生成、打印任务创建和状态维护。
- `web`: 负责订单上传、过滤列表、打印类型选择、PDF 预览和打印确认。
- `print-agent`: 第二阶段负责连接本地打印设备并执行打印任务。
- `sql`: 负责数据库初始化和表结构脚本。

## 技术约定

- 后端使用 Java 17、Spring Boot 3、MyBatis-Plus、MySQL 8。
- Excel 读取和打印设置使用 Apache POI。
- Excel 转 PDF 通过 LibreOffice 命令行完成。
- 前端使用 React、Vite、TypeScript、Ant Design。
- 数据库脚本默认按 MySQL 语法编写。

## 核心数据流

1. 上传 Excel 或图片订单原稿，保存到 `source_file`。
2. Excel 解析“订单”sheet，写入 `order_record`；图片写入待手动补充订单。
3. 订单列表按订单号、款号、客户、交期、识别状态过滤。
4. 点击打印后选择 `ORDER` 或 `PACKING`。
5. 后端只提取对应 sheet 生成 `print_preview` PDF。
6. 用户确认份数后创建 `print_task`。

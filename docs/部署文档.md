# 部署文档

## 环境要求

- JDK 17+
- Maven 3.8+
- Node.js 18+
- MySQL 8+
- LibreOffice

## 部署步骤

1. 执行 `sql/init.sql` 初始化数据库。
2. 执行 `sql/print_table.sql` 创建打印任务表。
3. 修改 `backend/src/main/resources/application.yml` 中的数据库连接。
4. 确认 `soffice` 可在命令行执行，或配置 `app.file-storage.libre-office-command` 为完整路径。
5. 构建并启动 `backend`。
6. 启动 `web` 进入订单工作台。
7. 第一阶段暂不需要启动 `print-agent`，只需要通过接口创建和更新打印任务。

## 后端构建

```powershell
cd D:\work\shoe-factory-assistant\backend
$env:JAVA_HOME="D:\develop\JAVA\jdk-17.0.18"
$env:Path="$env:JAVA_HOME\bin;$env:Path"
mvn clean package
java -jar target\shoe-factory-backend-0.1.0-SNAPSHOT.jar
```

## 前端启动

```powershell
cd D:\work\shoe-factory-assistant\web
npm.cmd install
npm.cmd run dev
```

Vite 会把 `/api` 代理到 `http://localhost:8080`。
